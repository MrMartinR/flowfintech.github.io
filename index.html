About The Project

This project implements a finance management application to track investments, p2p, Stocks, Shares, Forex, Crypto, budgets, retirements, Financial Independence Retire Early movement (FIRE), all stuff related to Personal Finance.

Packager Manager


macOS

via Homebrew

brew install yarn


Windows

via Node Package Manager (npm)

npm install -g yarn


Ubuntu

curl -sS https://dl.yarnpkg.com/debian/pubkey.gpg | sudo apt-key add -



echo "deb https://dl.yarnpkg.com/debian/ stable main" | sudo tee /etc/apt/sources.list.d/yarn.list


Note: DO NOT use npm on the project! (With the exception of installing Yarn on windows)
